Overall Job

If you are a web designer you will often :

- Interact with clients or use statistics to create an optimal website suited to user preferences

- Create plans for websites to determine what software must be created for and incorporated 

- Organize your website's information so it has a clean layout and proceed to add images/colours/animation
  to make the website presentable
  
- Create and test prototype websites to make sure all the code is functioning properly

Potential Job Titles 
- Web Designer or Web Developer
- "Program" Designer, E.x. UI/UX Designer
- Webmaster
- Manager of Web Services
- Internet Site Designer or Internet Site Developer
- Intranet Site Designer
- e-Business Web Site Developer
- e-Commerce Web Site Developer
